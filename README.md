# Server Stats

This is a Bash script that analyzes basic Linux server performance stats.

## Project URL

https://roadmap.sh/projects/server-stats

## Features

- Total CPU usage
- Total memory usage
- Total disk usage
- Top 5 processes by CPU usage
- Top 5 processes by memory usage
- OS version
- Uptime
- Load average

## How to Run

```bash
chmod +x server-stats.sh
./server-stats.sh
